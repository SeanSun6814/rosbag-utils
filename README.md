# Rosbag-utils

## Get started

### Installation

1. Clone this repo
2. `python3 app.py`

### Tutorial

[video](https://drive.google.com/file/d/1CQdt6Wb7p-Y6IWatlXA0cpNmh8SU0kE1/view?usp=sharing)

## Features

-   Filter topics from rosbags

-   Batch processing

-   Calculate trajectory length

-   Auto crop blank space at the beginning of rosbags
